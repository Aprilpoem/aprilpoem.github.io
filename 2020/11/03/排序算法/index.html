排序算法

对于长度为n的无序序列

约定左必右开

直接插入排序

概述：取无序的首部插入到前面有序序列中；

如何插入：从后往前和所取元素val比较，有序元素往后移array[j+1] = array[j]直到array[j] < val,

把val插入位置j+1,即array[j+1] = val

希尔排序

概述：多趟插入排序

  1. 按照步长选择得到多组序列，对他们进行插入排序；
  2. 减小步长（ gap = gap / 3 + 1 ）进行步骤1直到步长=0;

gap的选择也可以一直/2 直到gap==1

冒泡排序

冒泡排序：可以理解为每一趟的最大值像冒泡泡一样，一步步交换到固定的位置；

选择排序

概述：每次从无序的首部往后比较，选择最小的元素，和首部交换位置，直到有序的部分长度为n-1；

快速排序

核心步骤：

partition(array, start, end), return 下标

  1. 选pivot

    • 随机选，然后和第0个元素交换；
    • 直接选第0个元素；
    • 选三个元素(首，尾，中间)中的中间大值，和第0个元素交换。

  2. 交换（按坑填值比较好理解一点，注意left<right的条件）

    (1.) left=start，right=end-1, 选择pivot，空出第一个位置(left)的坑

    (2.) right从右往左比较，找到一个比pivot小的数，填入left的坑，此时right指向的位置是个坑；

    (3.) left从左往右比较，找到一个比pivot大的数，填入right的坑，此时left指向的位置空出来；

    (4.) 当left=right的时候跳出循环，此时这个位置一定是个坑，填入pivot的值

QuickSort(array, start, end)

递归函数，一次partition操作确定并返回pivot的排序位置mid，此时左侧为<=pivot的数；右侧为>pivot的数。

归并排序

核心步骤：

MergeSort( nums, tmp, start, end)

  1. 中心点 mid = ( start + end) / 2

  2. 递归调用MergeSort，排好序 nums[start: mid], nums[mid:end]

  3. 排好序再合并

    Merge( nums, tmp, start, mid, end)

    合并有序的nums[ start:mid ] 和 有序的nums[ mid:end ]

    tmp是额外的数组用来存合并之前的数组

不稳定的意义

以某种关键字的方式排序后，能不影响到其他关键字原来排序结果的方法就是稳定的

稳定性应用到数据库里会有要求吧
